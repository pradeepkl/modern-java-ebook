# Source Code

Companion runnable Java examples for the *Modern Java* ebook. Each chapter
directory maps to book listings generated by the pipeline (Agent 5).

**Requirements:** JDK 21+

## How to Compile and Run

### Flat chapters (1–4, 6–16)

Most examples are single-file listings in a flat package:

```bash
cd source-code/chapterXX
javac --enable-preview --release 21 *.java
java ClassName
```

Some listings use Java 9+ instance `main` or preview language features.
Sidecar `.java.warning` files record compile issues flagged during pipeline
validation.

### Chapter 5 (multi-module)

Chapter 5 is a runnable JPMS project. See [chapter05/README.md](chapter05/README.md)
for the module layout, listing paths, and build instructions:

```bash
cd source-code/chapter05
./compile.sh
```
